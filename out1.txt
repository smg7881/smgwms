                [화면 display]
                -. 고객재고속성 정보를 화면에 Display
                [고객]

                -. 조회조건값을 default로 setting 함.
                [입출고구분]

                -. 기본값 : ‘선택’ 사용자가 Combo box에서 선택 함
      화면 제어     -. 입출고구분 값을 기타코드에서 조회하여 콤보로 셋팅한다. (기타코드분류코드 ‘82’)
                [재고속성구분]

                -. 기본값 : ‘선택’ 사용자가 Combo box에서 선택 함
                -. 재고속성구분 값을 기타코드에서 조회하여 콤보로 셋팅한다. (기타코드분류코드 ‘101’)

                [사용여부]
                -. 기본값 : ‘Y’ 사용자가 Combo box에서 선택 함
                -. 로케이션구분 값을 기타코드에서 조회하여 콤보로 셋팅한다. ‘전체’포함 (기타코드분류코드 ‘06’)


      업무 규칙     -.


document        -.조회조건에 조회 하고자 하는 고객 코드 또는 명을 입력한 후, 찾기 버튼을 이용하여 원하는 고객을 선택한다.
                -.입츨고여부를 선택한다.

     이벤트 흐름     -.검색버튼을 클릭한다.
                -.조회조건에 해당하는 고객재고속성 목록이 DISPLAY된다.
     (순서 정의)
                -.고객재고속성 추가시 추가버튼을 click하여 새로운 Row에서 데이타를 입력하고 저장버튼을 click한다.
                -.고객재고속성 삭제시 선택 Check box를 click하여 삭제할 Data를 선택하고 삭제 버튼을 click 한다.
                - 고객재고속성 정보 변경시 변경하고자 하는 값을 변경한 후 저장버튼을 click한다.
